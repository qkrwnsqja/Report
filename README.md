# 코딩 테스트 웹 페이지 제작

본 프로젝트는 HTML5, CSS3, 그리고 반응형 웹 디자인 기술을 기반으로, 코딩 테스트 플랫폼을 소개하는 단일 페이지 웹사이트입니다. 중간고사 과제로 제출된 이 프로젝트는 실전 코딩 학습 환경을 시각적으로 구성하는 데 중점을 두었습니다.

## 프로젝트 주요 기능

- 코딩 테스트 개요 및 플랫폼 안내
- 최신 트렌드 및 기술 뉴스 제공
- 갤러리 섹션을 통한 시각 자료 표시
- 주요 수치 강조: 프로젝트 수, 사용자 수 등
- 사용자 행동 유도(Call to Action) 섹션 구성
- 연락처 정보 및 소셜 미디어 링크 포함
- 반응형 웹 구조로 다양한 기기에서 호환

## 사용 기술

- HTML5 시맨틱 마크업  
  `<header>`, `<section>`, `<footer>`, `<main>` 등을 이용한 구조적 웹 구성

- CSS3 스타일링  
  - 기본 스타일 초기화 및 레이아웃 정렬  
  - Flexbox를 활용한 콘텐츠 정렬  
  - 배경색, 텍스트 강조, 여백 및 그림자 처리

- 반응형 웹 디자인  
  - 미디어 쿼리를 통한 화면 크기 대응  
  - 모바일, 태블릿, PC 환경에 최적화된 구성
